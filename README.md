## ✨ 功能特点

- 🕐 **多任务定时抢课**：设定多门课程各自的开抢时间（精确到秒），到点自动执行
- 🔍 **三备份检索策略**：精确课号 → 模糊课名 → 去尾缀课名，逐级兜底
- 🔄 **自动切换分类Tab**：不同课程分类（主修/公共艺术/通识选修等）自动切换
- 💾 **本地持久化存储**：任务队列保存在油猴本地数据库，刷新/关闭浏览器不丢失
- 🗑️ **单项精确删除**：通过行号输入框指定删除，不用全部清空重填
- 🔍 **数据库可视化查看**：一键查看油猴内部数据库的原始内容
- 📥 **运行日志导出**：一键导出完整运行日志为TXT文件，便于排错
- ⚡ **防卡死自动重载**：网页假死6秒自动刷新恢复

## 📖 使用方法与步骤

1. 登录教务系统，进入 **自主选课** 页面。
2. 在右侧悬浮面板中：
   - 填写 **精确课号** 或 **模糊课名**（注意防空格）。
   - 选择该课程对应的 **分类**。
   - 设定该课程的 **抢课时间**。
3. 点击 **`+ 添加到上方列表`**。
4. **【至关重要】** 点击 **`💾 保存并提交`**（确保存盘列显示 **✅**，此时刷新或关闭网页数据才不会丢失）。
5. **【开始运行】** 点击 **`启动挂机监控任务`**，此时脚本才会真正开始托管运行。

## ⚠️ 核心注意事项（请务必仔细阅读）

为了保证脚本能正常跑通，请在操作时务必注意以下细节：

1. 🚫 **输入框绝对不能带空格**：
   - 填写**精确课号**（纯数字或字母）时，**前后和中间千万不能有空格**。
   - 填写**模糊课名**（中文汉字）时，也请仔细检查**不要混入空格**。若带有空格，系统会因字符不匹配而检索失败。
2. ⚙️ **必须手动点击“启动监控”**：
   - 保存好任务后，**一定要点击“启动挂机监控任务”按钮**！
   - 如果不点启动，脚本只会静止在保存状态，到点并不会自动执行。
3. ⏰ **检查时间格式与分类**：
   - 时间格式必须为 `HH:MM:SS`（例如 `09:00:00`），填错将无法准时触发。
   - 填写的【分类】必须和教务系统网页上的选项卡名称**完全一致**（如主修、通识选修等），否则脚本会进错板块。
4. 🗑️ **如何删除单项任务**：
   - 在下方的行号输入框中输入你想要删除的行号（对应列表里的 **`#`** 列数字）。
   - 点击 **`🗑️ 删除该行`** 按钮，即可即时删除并自动存盘。
